# Weather App Pro - Cosmic Edition üåå

A premium, glassmorphism-styled weather dashboard built with **Ionic**, **Tailwind CSS**, and **OpenWeatherMap API**.

This project features a "Bento Grid" layout, VisionOS-inspired glass effects, and a dynamic "Cosmic" color palette that adapts to weather conditions.

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## ‚ú® Features

- **Cosmic Aesthetic**: Deep void purple backgrounds with vibrant neon accents (Cyan, Magenta, Gold).
- **Bento Grid Layout**: A responsive, organized grid for data visualization (Hero, Map, Metrics).
- **Premium Glassmorphism**: High-quality blurs, gradient borders, and inner gloss effects (VisionOS style).
- **Interactive Maps**: Integrated Leaflet precipitation map.
- **Dynamic Backgrounds**: Shifts between Clear Day (Blue), Clear Night (Deep Purple), Rain (Navy), and more.
- **PWA Ready**: Installable on mobile and desktop.

## üöÄ Getting Started

### Prerequisites

- Node.js (v16+)
- NPM

### Installation

1.  **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/weatherapp-pro.git
    cd weatherapp-pro
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Start Development Server**

    ```bash
    npm run dev
    ```

4.  **Build for Production**
    ```bash
    npm run build
    ```
